# ecommerce Data project - 1
Exploratory Data Analysis of e-commerce sales data using python
Data set: E-Commerce Sales Performance Analysis
Source: https://www.kaggle.com/datasets/sidramazam/e-commerce-sales-performance-analysis/data
PSD : 03-04-2026

## Dataset
The dataset contains historical e-commerce transaction records including:
- Order Date
- Product Name
- Category
- Region
- Quantity
- Sales
- Profit
